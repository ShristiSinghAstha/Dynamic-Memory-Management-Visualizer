About the Project

This visualizer demonstrates how memory is allocated and managed using strategies like FIFO, LRU, First Fit, Best Fit, and Worst Fit. All wrapped in a clean, animated dashboard to aid understanding of memory operations.

Interactive Visualization

Watch as memory is dynamically allocated with real-time frame transitions and performance charts.

Allocation Strategies

Explore and compare the impact of multiple memory allocation strategies on system performance.

Key Features

  • Real-Time Updates

    Live updates of memory frames with animation and status indicators for hits and faults.

  • Multiple Strategies

    Supports FIFO, LRU, and other strategies with switchable modes and customizable speed.

  • Performance Insights

    Graphical insights into page hits and faults via Chart.js integration for clear understanding.
